# SkyExchange
**SkyExchange** - это фейк-биржа по трейдингу и покупки NFT. Трейди, покупай NFT и ставь их в качестве своего аватара.
# Технологии используемые в данном проекте:
<ul>
  <li>React</li>
  <li>TypeScript</li>
  <li>Zustand</li>
  <li>FastAPI</li>
  <li>NodeJs</li>
</ul>

**Данный проект создавался в рамках школьного проекта, данный код может использоваться свободно без каких либо ограничений.**
